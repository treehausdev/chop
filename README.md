# CHOP

Sample any song, finger-drum your own beats.

## Structure

```
chop/
├── web-app/    — Next.js web version (penny.codes/chop)
├── ios/        — Native iOS app (SwiftUI + AVAudioEngine)
└── README.md
```

## Web App

```bash
cd web-app
npm install
npm run dev
```

## iOS App

Open `ios/Chop.xcodeproj` in Xcode. Requires iOS 17+.
